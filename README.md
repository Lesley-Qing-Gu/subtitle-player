# Subtitle Player (SRT-based Text Playback Tool)

A lightweight web-based subtitle player that displays text according to SRT timecodes.

Designed for film studies, subtitle-based artworks, kinetic typography, and text-driven visual projects.

Supports desktop and mobile browsers.
